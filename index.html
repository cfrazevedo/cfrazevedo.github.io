Sobre mim

Sou um profissional multifacetado que atua em diferentes frentes na carreira de dados.

Ao longo da minha trajetória profissional, sempre tive um olhar voltado para análise, interpretação e resolução de problemas. Esse caminho me conduziu naturalmente para a área de dados, onde encontrei não apenas uma carreira, mas um campo em constante evolução que conecta tecnologia, estratégia e negócios. Apesar de ter formação acadêmica com mestrado em outra área, realizei uma transição de carreira bem sucedida.

Nos últimos anos, venho me aprofundando em temas que envolvem todo o ciclo de dados, desde a elaboração do projeto, passando por engenharia de dados, de analytics, análise e ciência de dados. Participei recentemente do Programa Lighthouse da Indicium, que consolidou minha visão de um profissional Data Full Stack: alguém preparado para transitar entre diferentes etapas do ecossistema de dados, entendendo como cada peça se conecta. Essa experiência, junto às ferramentas que domino (como Python, SQL, Databricks, dbt, Power BI, Streamlit, entre outras), ampliou ainda mais minha capacidade de gerar impacto.

Hoje, meu foco é avançar nessa carreira, colaborando com equipes que buscam transformar dados em vantagem competitiva. Estou aberto a novas conexões, oportunidades e desafios que me permitam aplicar minhas habilidades e continuar aprendendo nesse universo fascinante.

Projetos

AutoMap - Indicium

Este projeto contém uma ferramenta para automatizar o mapeamento e diagnóstico de modelos de dados em Power BI dashboards. A ferramenta analisa as tabelas geradas por ferramentas de terceiros e mapeia o fluxo de uso de tabelas, colunas e métricas no seu dashboard.


Ferramentas utilizadas:
• Power BI
• DAX Studio | Power BI Helper
• Python | Pandas

Adventure Works - Programa Lighthouse

Este projeto contém um ELT pipeline desenvolvido de ponta a ponta para a Adventure Works (fictícia), utilizando Databricks para orquestração e ingestão de dados, dbt para transformação e modelagem analítica, e uma camada de machine learning para análises preditivas. Acessando pela figura, terá o resumo do desenvolvimento da modelagem preditiva. O repositório deste projeto está disponível aqui.


Ferramentas utilizadas:
• Databricks | dbt
• Python | Pandas | Matplotlib | Seaborn
• Prophet | Scikit-learn

Desafio Programa Lighthouse - Caso BanVic

Este projeto apresenta um relatório com análises detalhadas dos dados do BanVic (banco fictício), com indicadores-chave de desempenho (KPIs) e recomendações práticas para o uso estratégico de dados na tomada de decisões. Foi realizada uma análise exploratória do banco de dados, gráficos foram confeccionados para realizar análises temporais, entendendo seus padrões. Acesse o desenvolvimento de EDA e gráficos aqui.


Ferramentas utilizadas:
• Python | Colaboratory
• Pandas | Matplotlib | StatsModels

World Mining Dashboard - Streamlit

Este dashboard apresenta dados globais de mineração, explorando recursos minerais de continentes, blocos econômicos e regiões do mundo todo por meio de gráficos interativos. O meu objetivo foi apresentar insights detalhados sobre categorias de matérias-primas — como metais ferro-ligas, não ferrosos e outros — juntamente com estatísticas sobre minerais e países específicos.


Ferramentas utilizadas:
• Python | Streamlit
• Pandas | Geopandas | Plotly | Folium

Dashboard de vendas - Streamlit

Este é um dashboard criado para a análise visualização de vendas de uma rede de loja fictícia presente em varias cidades e regiões do Brasil. Ele foi desenvolvido usando uma ferramenta de visualização do Python chamada Streamlit. Ele é capaz de gerar paginas e dashboards online sem usar HTML/CSS, somente linguagem Python. O objetivo aqui é demonstrar a capacidade e as ferramentas que podem ser usadas no Streamlit para criar dashboards. O repositório deste projeto está disponível aqui.


Ferramentas utilizadas:
• Python | Streamlit
• Pandas | Plotly | Requests
